## Project Overview

Co-authored the Azure Advisor cost optimization workbook in collaboration with the Azure Advisor product group to help customers identify cost-saving opportunities.

### Key Achievements

- **Product Group Collaboration** - Worked directly with Azure Advisor team
- **Wide Adoption** - ~1,800 monthly active users
- **Cost Savings Impact** - Helping customers optimize their Azure spend

### Workbook Features

- **Resource Optimization** - Identify underutilized resources
- **Reserved Instance Recommendations** - RI purchasing guidance
- **Right-sizing Suggestions** - VM and resource sizing recommendations
- **Idle Resource Detection** - Find and clean up unused resources
- **Cost Trends** - Visualize spending patterns over time

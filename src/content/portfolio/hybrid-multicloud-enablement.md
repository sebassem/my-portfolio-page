## Project Overview

Led the creation of field content and tools to support delivering customer engagements around security, governance, and management of hybrid and multi-cloud solutions using Azure Arc.

### Key Achievements

- **177 Deliveries in FY23** - Field content delivered to customers worldwide
- **Comprehensive Tooling** - Built tools to accelerate customer engagements
- **SME Role** - Azure Arc and Azure Advisor SME in the Azure WW community

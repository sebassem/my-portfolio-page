## Project Overview

Built the Azure Governance Visualizer accelerator, simplifying deployment and automating upgrades to help customers gain better visibility into their Azure governance posture.

### Key Features

- **Simplified Deployment** - One-click deployment experience
- **Automated Upgrades** - Keep the visualizer up-to-date automatically
- **Governance Insights** - Comprehensive view of Azure governance
- **Policy Compliance** - Track policy compliance across subscriptions

### What It Visualizes

- Management Group hierarchy
- Subscription organization
- Policy assignments and compliance
- Role assignments (RBAC)
- Resource providers
- Azure AD/Entra ID integration

### Benefits for Customers

- **Better Visibility** - Understand your Azure governance at a glance
- **Compliance Tracking** - Monitor policy compliance across the organization
- **Audit Readiness** - Generate reports for audits and reviews
- **Troubleshooting** - Quickly identify governance issues

### Technologies

- PowerShell
- Azure DevOps / GitHub Actions
- Azure Resource Graph
- HTML/CSS/JavaScript
- Mermaid Diagrams

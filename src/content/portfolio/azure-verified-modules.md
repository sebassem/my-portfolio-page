### What is Azure Verified Modules?

Azure Verified Modules is a Microsoft initiative to provide pre-built, tested, and supported Infrastructure-as-Code modules that follow best practices for:

- Security
- Reliability
- Operational excellence
- Performance efficiency
- Cost optimization

### Modules Contributed

Each module follows AVM standards including:
- Comprehensive documentation
- Unit and integration tests
- Example deployments
- Telemetry and versioning

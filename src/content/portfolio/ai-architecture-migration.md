## Project Overview

Designed the architecture of an AI solution for a major enterprise customer, facilitating their migration from AWS/OpenAI to the Azure AI platform.

### Key Achievements

- **Enterprise-scale Architecture** - Designed end-to-end AI solution architecture for a large customer
- **Cloud Migration** - Led the transition from AWS/OpenAI to Azure AI platform
- **Infrastructure-as-Code** - Built the entire solution using IaC principles
- **Security Best Practices** - Implemented comprehensive security controls
- **Resiliency Patterns** - Ensured high availability and disaster recovery

### Technologies Used

- Azure OpenAI Service
- Azure AI Services
- Bicep / ARM Templates
- Azure Virtual Networks
- Azure Key Vault
- Azure Monitor

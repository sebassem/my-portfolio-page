## Project Overview

Built automated tools to help customers assess their workloads against the Azure Well-Architected Framework's best practices, significantly reducing the time and cost of assessment deliveries.

### Key Achievements

- **40% Reduction** - Reduced time and cost of Well-Architected assessment deliveries by ~40%
- **Automated Tooling** - Built tools for automated workload assessment
- **Training & Enablement** - Conducted training sessions, webinars, and bootcamps
- **Field Empowerment** - Enabled field resources and customers to deliver assessments

### Impact Metrics

| Metric | Value |
|--------|-------|
| Time/Cost Reduction | ~40% |
| Delivery Method | Automated Tools |
| Enablement | Training, Webinars, Bootcamps |

### Tools Developed

The automated assessment tools help customers:
- **Evaluate workloads** against Well-Architected Framework pillars
- **Identify gaps** in their architecture
- **Prioritize improvements** based on impact
- **Generate reports** automatically

### Well-Architected Pillars Covered

1. **Reliability** - Ensuring workloads recover from failures
2. **Security** - Protecting applications and data
3. **Cost Optimization** - Managing costs effectively
4. **Operational Excellence** - Running and monitoring systems
5. **Performance Efficiency** - Using resources efficiently

### Training & Enablement

Contributed to empowering field resources and customers through:
- Multiple training sessions
- Public webinars
- Training bootcamps
- Documentation and guides

### Technologies Used

- Azure Well-Architected Framework
- PowerShell / Azure CLI
- Azure Resource Graph
- Azure Advisor
- Azure Policy

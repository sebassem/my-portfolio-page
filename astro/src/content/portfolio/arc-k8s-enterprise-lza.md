## Project Overview

Worked as part of a virtual team to create the Azure Arc-enabled Kubernetes Enterprise Scale Landing Zone Accelerator, leading the cost governance design area and building the sandbox reference architecture.

![Screenshot showing the K8s landing zone accelerator architecture](https://images.seifbassem.com/images/portfolio/lza-accelerators/lza-kubernetes.png)


### Key Achievements

- **Microsoft Build 2022 Announcement** - Solution announced at Microsoft's flagship developer conference
- **Cost Governance Lead** - Led the design area for cost management, detailing the best practices for optimizing costs when deploying Azure Arc-enabled Kubernetes at scale like cost allocation strategies, budget management and alerts, resource optimization recommendations
- **Sandbox Development** - Contributed to building the hands-on sandbox solution for the product, including solution automation via Bicep and PowerShell

![Screenshot showing Arcbox for DevOps architecture](https://images.seifbassem.com/images/portfolio/lza-accelerators/lza-kubernetes-arcbox-devops.png)

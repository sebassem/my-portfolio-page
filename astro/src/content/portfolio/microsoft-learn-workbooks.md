## Project Overview

Authored a Microsoft Learn Module about Azure Monitor Workbooks, including an interactive sandbox environment where learners get hands-on experience with the service.

### Key Achievements

- **Complete Learning Module** - End-to-end educational content on Azure Workbooks
- **Interactive Sandbox** - Deployed environment with sample application and database
- **Hands-on Experience** - Learners practice in a real Azure environment
- **Adoption Driver** - Designed to drive more adoption of Azure Monitor Workbooks

### Module Features

#### Interactive Sandbox
Learners receive a fully deployed environment including:
- Sample application
- Database with realistic data
- Pre-configured Azure Monitor resources
- Guided exercises

#### Learning Objectives
The module teaches learners to:
- Create and customize Azure Workbooks
- Visualize data from multiple sources
- Build interactive dashboards
- Use parameters and filters
- Share workbooks across teams

### Impact

This module helps drive adoption of Azure Monitor Workbooks by:
- Lowering the barrier to entry
- Providing practical, hands-on experience
- Demonstrating real-world scenarios
- Building confidence through guided exercises

### Technologies Covered

- Azure Monitor Workbooks
- Kusto Query Language (KQL)
- Azure Log Analytics
- Azure Application Insights
- Azure Resource Graph
